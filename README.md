This project is a MERN stack application that allows users to type a message, generate an AI response, preview the result live, and save the data to MongoDB.

---

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MongoDB Atlas
- AI API: OpenRouter
- Deployment:
  - Frontend: Vercel
  - Backend: Render

---

## Features
- Type a message from the dashboard
- Generate AI response
- Live preview 
- Save prompt and response to MongoDB
- REST API integration

---

## Environment Variables

Create a `.env` file in the backend folder:

```env
MONGO_URI=your_mongodb_connection_string
OPENROUTER_API_KEY=your_openrouter_api_key
PORT=http:5000
---
